# SmartSpend

A Budget management website made using **React JS, Tailwind CSS, Daisy UI, MongoDB, and Express JS**

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install foobar.

```bash
npm install
```

## Run the project

```bash
npm run dev
```
